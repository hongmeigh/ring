夜间模式

现在我们提供一个夜间模式，你只需要在 body 或者 .page 或者 .content 上加上 .theme-dark。它和其中所有子元素都会变成夜间模式。

你也可以单独给 .bar 加上 .theme-dark，这样可以单独使标题栏或者工具栏变成夜间模式。

夜间模式最大的区别是他的背景变成了黑色，而前景色变成了白色。

夜间模式还处在测试阶段，可能会有某些组件在夜间模式下显示不正常。有任何问题都可以及时在旺旺群中反馈。

下一节是介绍颜色主题的用法，颜色主题主要是改变前景色，可以和 .theme-dark 结合使用以在夜间模式下实现不同的颜色。

颜色主题

不同于夜间模式，颜色主题只改变组件的“主色”。你只需要在body 或者 .page 或者 .content 上加上 theme-{color} 就可以使它的所有子元素的主色改变。

现在有如下几种颜色可用:

  • .theme-pink 粉色主题
  • .theme-green 绿色主题
  • .theme-yellow 黄色主题

你可以在右侧的示例中点击按钮来切换不同的颜色主题。

注意颜色主题并不会影响所有的颜色，他只会影响组件的主色。因此有些组件没有使用到主色，因此对他们没有影响。

颜色主题还处在测试阶段，可能会有某些组件在颜色主题下显示不正常。有任何问题都可以及时在旺旺群中反馈。
